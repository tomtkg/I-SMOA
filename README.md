# I-SMOA
Iterative Supervised Multi-objective Optimization Algorithm

## How to use
ISMOA.m is main file. You can execute ISMOA.m like 
```MATLAB
matlab -batch ISMOA
```

Similar code will be implemented in PlatEMO.
